# FreeAndOpenCell

A free and open-source FreeCell solitaire game built with SolidJS.

License: AGPL-3.0-or-later

## Development

Prerequisites: Node 22, pnpm

```bash
pnpm install
pnpm dev
pnpm test
pnpm build
```
