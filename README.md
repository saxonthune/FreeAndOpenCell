# FreeAndOpenCell

A free and open-source implementation of FreeCell.

Created using spec-driven development practices. Learn more at [computation.saxon.zone](https://computation.saxon.zone). Specs maintained with [carta](https://github.com/saxonthune/carta).

Live demo on [GitHub pages](https://saxonthune.github.io/FreeAndOpenCell/).

## Development

Prerequisites: Node 22, pnpm

```bash
pnpm install
pnpm dev
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```
