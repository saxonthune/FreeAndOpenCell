# Vision

FreeAndOpenCell is a playable FreeCell game shipped as a static web app, built with SolidJS.

## Two Goals

1. **The game** — a working FreeCell implementation that runs entirely in the browser, deployable as static files.
2. **The method** — use this project as a testbed for spec-driven development with [carta](https://github.com/saxonthune/carta) and related tools. Specs live in `.carta/`; the code is built from them.

The second goal is the more interesting one. FreeCell is the vehicle, not the destination — it was chosen because the rules are well-known, the scope is bounded, and there's no ambiguity about what "done" looks like for the game itself. That leaves the experiment free to focus on the workflow.
